# Matrizes

Exemplo:
```java
package exemplo;
public class ExemploMatrizes {
    public static void main(String[] args) {
        // Declarando e inicializando uma matriz 3x3
        int[][] matriz = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    // Acessando elementos da matriz
    System.out.println("Elemento [0][0]: " + matriz[0][0]); // 1
    System.out.println("Elemento [1][2]: " + matriz[1][2]); // 6
    System.out.println("Elemento [2][1]: " + matriz[2][1]); // 8
    }
}
```
