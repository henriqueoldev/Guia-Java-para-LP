# Variáveis Locais

Exemplo:
```java
package exemplo;
public class ExemploLocal {
    public static void main(String[] args) {
        somar();
    }
    public static void somar() {
        // Variáveis locais
        int a = 10;
        int b = 20;
        int soma = a + b;
        System.out.println("A soma é: " + soma);
    }
}
```
