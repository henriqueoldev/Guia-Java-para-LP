# Modularização - Função

```java
package exemplo;
public class ExemploFuncao {
    public static void main(String[] args) {
        int resultado = somar(5, 7);
        System.out.println("O resultado da soma é: " + resultado);
    }
    public static int somar(int a, int b) {
        return a + b;
    }
}
```
