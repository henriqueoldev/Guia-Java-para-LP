# Estrutura Condicional - else

```java
package exemplo;
public class ExemploElse {
    public static void main(String[] args) {
        int numero = -5;
        if (numero >= 0) {
            System.out.println("Número positivo ou zero.");
        } else {
            System.out.println("Número negativo.");
        }
    }
}
```
