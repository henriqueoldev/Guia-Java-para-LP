# Estrutura de Repetição - para cada (for-each)

```java
package exemplo;
public class ExemploForEach {
    public static void main(String[] args) {
        int[] numeros = {1, 2, 3, 4, 5};
        for (int numero : numeros) {
            System.out.println("Contador: " + numero);
        }
    }
}
```
