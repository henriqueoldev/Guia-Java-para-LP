# Modularização - Procedimento

```java
package exemplo;
public class ExemploProcedimento {
    public static void main(String[] args) {
        exibirMensagem("Renan");
        exibirMensagem("Habache");
    }
    public static void exibirMensagem(String nome) {
        System.out.println("Olá, " + nome + "! Bem-vindo ao Java.");
    }
}
```
